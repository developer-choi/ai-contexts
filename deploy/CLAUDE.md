## 완료 후 재확인

- 사용자가 요청한 작업을 완료하면, 산출물을 사용자의 원래 지시와 1:1로 대조하여 오류 0개가 나올 때까지 검증한 뒤 제출한다.
- 모든 형태의 작업에 해당한다: plan mode 계획 실행, 스킬 호출, 프롬프트로 직접 지시한 것, 여러 항목을 나열한 요청 등.
- 예: 피그마 링크를 받고 마크업한 경우, 피그마와 마크업 결과물을 1:1 대조하여 디자인 토큰·레이아웃·텍스트 등이 모두 일치하는지 확인한 뒤 제출한다.
- 단, 사용자가 명시적으로 무언가를 대조하라고 지시한 경우에는 이 자동 재확인을 적용하지 않는다 (사용자 지시가 곧 검증이므로 중복 방지).
