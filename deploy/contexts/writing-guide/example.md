# 문장 교정 사례

AI가 작성한 문장과 사용자가 확정한 문장을 기록합니다.
같은 실수를 반복하지 않기 위한 참고 자료입니다.

---

## 사례 1: "덜어주는 UX"

**Bad (AI)**

```
마우스 클릭이나 모바일 터치 한 번을 줄여 사용자의 불편함을 덜어주는 UX입니다.
```

```
페이지에 진입하자마자 바로 타이핑할 수 있는 것이 좋은 UX라고 생각합니다.
```

- "불편함을 덜어주는 UX" — 의미가 모호하고 주어-목적어 관계가 어색함
- "좋은 UX라고 생각합니다" — 뭘 말하려는 건지 전달이 안 됨

**Result (사용자 확정)**

```
이런 작은 불편함을 줄여주는 것이 좋은 UX 중 하나라고 생각합니다.
```

- "불편함을 줄여주는 것" — 동작이 명확
- "좋은 UX 중 하나" — 단정이 아닌 의견으로 톤 조절
