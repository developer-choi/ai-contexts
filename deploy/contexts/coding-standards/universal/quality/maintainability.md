# 수정하기 쉬운 코드

응집도가 높은 코드
- Coincidental cohesion
- Logical cohesion 등
- 응집도의 종류를 기반으로 피드백 할것.

결합도가 낮은 코드
- Content coupling
- Common coupling 등
- 결합도의 종류를 기반으로 피드백 할것.

## 파일·폴더 크기 불균형은 구조 재검토 신호

특정 파일이나 폴더가 주변 대비 비대해졌다면, 폴더 컨벤션이나 모듈 분할을 재검토할 계기로 삼습니다.

- 비대함 자체가 문제는 아닙니다. 응집도·결합도 측면에서 적절하다면 단지 해당 도메인이 큰 것뿐입니다.
- 그러나 대부분의 경우, 비대해진 파일·폴더에는 반영되지 않은 하위 분류가 숨어 있습니다. 크기 불균형을 발견하면 누락된 분류가 없는지 확인하세요.
