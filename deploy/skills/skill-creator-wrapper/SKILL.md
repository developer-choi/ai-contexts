# Skill Creation Guide

## 사전 확인

skill-creator 플러그인이 설치돼 있는지 확인한다. 스킬 목록에 `skill-creator:skill-creator`가 없으면 사용자에게 안내한다:

> skill-creator 플러그인이 필요합니다. Claude Code에서 `/install-plugin https://github.com/anthropics/claude-code-plugins` 로 설치하세요.

## 규칙

skill-creator 플러그인을 사용하되, 이 프로젝트에서는 [guide.md](guide.md)의 규칙을 추가로 따른다.
