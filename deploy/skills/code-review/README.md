# Code Review

코드 변경사항을 리뷰하고 이슈 목록을 산출하는 스킬입니다. 수정은 하지 않습니다.

## 왜 만들었나요?

코드 리뷰에서 같은 지적을 반복하고 싶지 않았습니다. 자주 나오는 지적은 코딩 컨벤션으로 문서화하고, AI가 그 컨벤션을 기반으로 리뷰하도록 만들었습니다. 컨벤션에 없는 문제도 전문가 판단으로 짚습니다.

## 사용 맥락

- **내 코드 점검**: workflow step-6에서 호출. multi-agent 리뷰 (convention ×N + advanced)
- **남의 PR 리뷰**: 독립 호출. 단일 리뷰 후 사용자가 pr-comment-write로 연결 여부 결정
