# Doc Router

구글 문서에 쌓아둔 필기를 활용 가능한 형태로 분류하는 스킬입니다.

## 왜 만들었나요?

그동안 공부한 내용을 구글 문서에 필기해왔습니다. 하지만 구글 문서에 쌓아두기만 하면 나중에 활용하기 어렵습니다. 정리하는 것 자체도 시간이 오래 걸렸습니다.

이 스킬은 구글 문서의 필기를 읽고, 실제로 활용할 수 있는 곳으로 분류합니다. 학습 콘텐츠는 [knowledge-archive](https://github.com/developer-choi/knowledge-archive)로, AI에게 내릴 지시는 ai-contexts로, 코딩 팁은 dsa-playground로. 어디에도 속하지 않는 내용은 제외하고, 애매한 내용은 사용자에게 확인을 받습니다.

> 자세한 내용은 [SKILL.md](SKILL.md)를 참고하세요.
