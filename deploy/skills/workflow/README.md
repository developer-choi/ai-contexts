# Workflow

AI와 함께 효율적으로 코드를 작성하기 위한 단계별 프로세스입니다.

## 진행 흐름

1. **배경 파악 및 문제 정의** — 프로젝트 컨텍스트 수집
2. **PR 분할 전략 수립** — 작업 범위를 PR 단위로 분할
3. **핵심 과제 정의 및 기술 전략 수립** — PR별 목표와 접근 방식 결정
4. **구현 방침 상세화** — 세부 구현 계획
5. **구현** — 코드 작성
6. **최종 점검** — 결과물 검증
7. **PR 본문 작성** — 리뷰어를 위한 PR 설명
8. **워크플로우 회고** — 프로세스 개선점 도출

Step 1~2는 초기 1회, Step 3~8은 PR마다 반복합니다.

## 폴더 구조

| 경로 | 설명 |
|------|------|
| `SKILL.md` | AI 행동 가이드 (스킬 호출 시 로드) |

| `template/` | 산출물 양식 (context-setup.md, codebase-audit.md) |
