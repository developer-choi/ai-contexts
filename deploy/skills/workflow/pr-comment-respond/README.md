# PR Comment Respond

내 PR에 달린 리뷰 코멘트를 분석하고 대응 전략을 세우는 스킬입니다.

## 왜 만들었나요?

코멘트를 받으면 단순히 고치는 것보다 중요한 게 있습니다. 내가 몰라서 틀린 건지, 실수한 건지, 의견 차이인지에 따라 대응이 달라집니다.

이 스킬은 코멘트를 네 가지로 분류합니다.

- **모르는 것** — 공식문서 등으로 학습이 필요합니다
- **실수/오류** — 재발 방지 조치가 필요합니다
- **의견 차이** — 양쪽 근거를 정리하고 수용할지 반박할지 판단합니다
- **단순 수정** — 바로 고칩니다

모르는 것은 학습 후 기록하고, 실수는 원인을 분석해서 스킬이나 규칙에 반영합니다. 같은 코멘트를 두 번 받지 않기 위해서입니다.

> 자세한 내용은 [SKILL.md](SKILL.md)를 참고하세요.
