# 요구사항 리뷰

## 역할

- **AI**: 요구사항을 비판적으로 분석하고, 발견한 문제를 사용자에게 설명합니다. 모호한 부분을 짚고, 엣지케이스를 선제시하며, 개선안을 제시합니다. 단정하지 않고 장단점을 함께 설명합니다.
- **사용자**: AI가 제시한 내용을 확인/판단합니다. 추가 자료를 제공합니다.

## 입력 타입

argument로 입력 타입을 받는다. 입력 타입은 조합 가능하다. 입력 타입에 따라 두 가지 절차 중 하나를 따른다.

### 페이지 단위 리뷰 (planning, design)

페이지 단위로 자료를 받아 반복 분석한다. 아래 페이지 단위 리뷰 루프를 따른다.

| 입력 타입 | 가이드 | 설명 |
|-----------|--------|------|
| planning | [planning/guide.md](planning/guide.md) | 기획서 리뷰 |
| design | [design/guide.md](design/guide.md) | 디자인 시안 리뷰 |

### 독립 절차 (recruitment)

페이지 단위 반복이 아닌, 가이드가 정의한 자체 절차를 처음부터 끝까지 따른다.

| 입력 타입 | 가이드 | 설명 |
|-----------|--------|------|
| recruitment | [recruitment/guide.md](recruitment/guide.md) | 채용과제 요구사항 리뷰 |

### 가이드 슬롯

각 가이드가 정의하는 슬롯:

| 슬롯 | 설명 | 페이지 단위 리뷰에서의 역할 |
|------|------|---------------------------|
| 준비 방식 | 리뷰 시작 전 수행할 것 | 루프 진입 전에 실행 |
| 입력 자료 | 사용자에게 요청할 자료 | 루프 1단계에서 참조 |
| 체크리스트 | 분석 시 적용할 체크리스트 | 루프 2단계에서 적용 |
| 산출물 | 분석 결과 파일 | 루프 3~4단계에서 기록 |
| 분석 과정 | 입력 타입별 분석 절차 | 루프의 페이지별 단계(1~2단계)를 구체화하고, 루프 종료 후 단계(공통화 검토, 종합)를 정의 |

독립 절차(recruitment)는 페이지 단위 리뷰 루프를 사용하지 않는다. 가이드가 정의한 자체 절차(공동 탐색 → 평가 기준 추론 → 프로젝트 결정 → 모호한 부분)를 처음부터 끝까지 따른다. 슬롯은 가이드 내부에서 자체적으로 참조된다.

### 입력 타입 조합 규칙

- planning + design: design은 planning의 산출물(page.md, global.md)이 존재하는 상태에서만 수행한다. planning이 선행되어야 하는 이유는 실행 순서가 아니라 산출물 의존성이다 — 이전 세션에서 planning을 완료했다면 design만 단독 수행할 수 있다. 종합 단계는 마지막에 한 번만 수행하며, planning과 design 양쪽의 종합을 통합 처리한다.
- recruitment + design: 채용과제에 디자인 시안(figma 등)이 포함된 경우 둘 다 수행한다. recruitment를 먼저 완료한 뒤 design을 수행한다.

## 페이지 단위 리뷰 루프

planning, design에 적용된다. 입력 타입별 준비가 끝나면, 아래 사이클을 페이지마다 반복한다. 각 가이드의 슬롯을 참조하여 실행한다.

### 1. 자료 받기

사용자로부터 리뷰 대상 자료를 받습니다. 구체적 요청 방식은 가이드의 준비 방식 슬롯을 따릅니다.

### 2. 같이 분석

가이드의 체크리스트 슬롯을 기반으로 AI와 사용자가 같이 분석합니다. 구체적인 분석 과정은 가이드의 분석 과정 슬롯이 이 단계를 구체화(override)합니다.

### 3. 논의

`/discussion`을 사용하여 사용자와 대화하며 미결 항목의 결론을 냅니다.

- 확정된 내용 → 산출물에 기록
- 사용자가 판단할 수 없는 항목 → TODO로 분류
- 논의가 끝나지 않은 항목 → "미확인"으로 표기

### 4. 확정

논의 결과를 산출물에 반영합니다. 사용자가 수정한 말투나 표현은 다듬지 않습니다. 사용자에게 산출물을 확인하라고 안내합니다. 수정할 부분이 있으면 반영하고, 없으면 다음 자료로 넘어갑니다.

## 회고
