# 서비스 분석 가이드

채용과제의 대상 서비스를 분석하여 `/plan/background/service-analysis.md`로 산출한다.

## 필수 분석 항목

### 주 연령층

- 서비스의 주요 사용자 그룹별 연령대와 특징
- 공식 통계가 없으면 이용 패턴 기반으로 추정

### 서비스가 제공하는 가치

- 사용자 유형별로 이 서비스가 해결하는 문제
- 사용자가 이 서비스를 선택하는 이유

### 비즈니스 모델

- 수익원과 과금 구조
- 수익 배분 구조 (해당하는 경우)

### 주요 사용자 이용 FLOW

- 사용자 유형별 핵심 이용 흐름
- 진입부터 목적 달성까지의 주요 단계
