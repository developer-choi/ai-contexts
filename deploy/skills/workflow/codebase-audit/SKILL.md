# Codebase Audit

사용자가 지정한 파일/폴더를 탐색하여 기존 코드의 영향도를 분석합니다.
Plan mode로 진행하세요 — 모듈 간 의존성과 파일 구조를 직접 탐색해야 합니다.

## 분석 항목

- **가용 자원 탐색**: 이미 구현된 것 찾기
- **수정 위치 특정**: 어느 파일 어느 라인
- **중복 방지**: 재사용 방법 고민
