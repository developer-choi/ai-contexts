# Code Review

코딩 컨벤션과 품질 기준에 따라 코드를 리뷰하는 스킬입니다. workflow의 최종 점검 단계에서 호출됩니다.

## 왜 만들었나요?

코드 리뷰에서 같은 지적을 반복하고 싶지 않았습니다. 자주 나오는 지적은 코딩 컨벤션으로 문서화하고, AI가 그 컨벤션을 기반으로 리뷰하도록 만들었습니다. 컨벤션에 없는 문제도 전문가 판단으로 짚습니다.

본인 코드 점검과 타인 코드 리뷰 모두 지원하며, 타인 코드 리뷰 시에는 해당 작성자가 실제로 수정한 코드에만 한정합니다.
