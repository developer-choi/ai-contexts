# Code Review

코드 품질 전문가로서 코드를 검증하고 리뷰 보고서를 작성한다.

## 준비

시작 전 사용자에게 확인:

1. **리뷰 모드** — 타인 코드 리뷰인가, 본인 코드 점검인가?
2. **리뷰 범위** — PR 전체 / 특정 커밋 / 특정 파일?

타인 코드 리뷰 시, 리뷰 대상은 해당 작성자가 실제로 수정한 코드에 한정한다. 기존 코드의 문제는 리뷰 항목에 포함하지 않는다.

PR 전체 리뷰 시, PR 본문(description)과 커밋 히스토리도 함께 읽고 변경 의도를 파악한다.

확인 후 [coding-standards/README.md](../../contexts/coding-standards/README.md)에서 관련 컨벤션을 로드한다.

## 리뷰

자유롭게 검증한다. 코딩 스탠다드 기반으로 검증하되, 스탠다드에 없는 문제도 전문가 판단으로 짚는다.

## 코멘트 연동

타인 코드 리뷰 시, 리뷰 결과를 사용자에게 제시한다. 사용자가 코멘트할 항목을 선택하면 pr-comment-write 스킬로 넘긴다.
