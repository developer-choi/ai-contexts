# PR Comment Write

다른 사람의 PR에 코멘트를 달기 전에, 내용을 검증하고 말투를 다듬어주는 스킬입니다.

## 왜 만들었나요?

PR 코멘트는 공개적이고 기록에 남습니다. 잘못된 지적은 신뢰를 떨어뜨리고, 날카로운 말투는 관계를 해칩니다.

이 스킬은 코멘트를 올리기 전에 세 가지를 체크합니다.

- **PR 본문 대조** — 지적하려는 내용이 이미 PR 본문에 설명돼 있지 않은지
- **변경 범위 확인** — 지적하려는 코드가 이 PR에서 수정된 건지, 기존부터 있던 건지
- **기술적 정확성** — 지적이 틀렸거나 근거가 약하지 않은지

문제가 발견되면 코멘트를 작성하지 않고 먼저 알려줍니다. 검증을 통과하면 담백하되 부드러운 톤으로 코멘트를 작성합니다.

> 자세한 내용은 [SKILL.md](SKILL.md)를 참고하세요.
