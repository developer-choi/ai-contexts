# 프론트엔드 엔지니어 (경력)

## 회사 소개

랭디는 **온라인 1:1 수업 플랫폼**을 운영하는 에듀테크 스타트업입니다.
선생님과 학생이 화상 통화로 연결되는 수업 환경을 만들고, 수업 품질을 실시간으로 모니터링·개선하는 도구를 만들고 있습니다.

현재 누적 회원 30만 명, 월 진행 수업 12만 건 규모로 성장하고 있으며, 안정성과 사용성을 동시에 끌어올려 줄 프론트엔드 엔지니어를 찾고 있습니다.

## 합류하시면 함께 할 일

- **수업 통화 모니터링 어드민** — 운영팀이 실시간으로 수업 품질을 확인하고 이슈에 대응할 수 있는 내부 도구
- **학생 / 선생님 웹앱** — 수업 예약, 진행, 결제, 후기 관리
- **공통 디자인 시스템** — 여러 프로덕트가 공유하는 컴포넌트 라이브러리

## 주요 업무

- 어드민 / 웹앱의 신규 기능 설계 및 구현
- 디자이너·백엔드·QA와 협업하여 화면 단위 요구사항을 제품으로 만들기
- 컴포넌트 단위 재사용성과 확장성을 고려한 코드 작성
- 사용자 경험과 성능을 지속적으로 측정·개선

## 자격 요건

- React 기반 웹 서비스 실무 개발 경험 **3년 이상**
- TypeScript에 대한 충분한 이해와 실무 활용 경험
- HTML / CSS 시맨틱과 접근성에 대한 기본기
- Git 기반 협업 경험 (브랜치 전략, 코드리뷰)
- 디자인 시안을 기반으로 한 화면 구현 경험

## 우대 사항

- Next.js (App Router) 실무 경험
- Feature-Sliced Design 등 **레이어 기반 폴더 구조** 적용 경험
- 디자인 시스템 / 컴포넌트 라이브러리 설계 경험
- 실시간 데이터 (WebSocket, SSE) 다룬 경험
- 테스트 코드 작성 경험 (Jest, Playwright 등)
- 어드민 / 내부 도구 개발 경험

## 기술 스택

- **언어**: TypeScript
- **프레임워크**: Next.js (App Router), React 19
- **스타일링**: Tailwind CSS
- **상태관리 / 데이터**: TanStack Query, Zustand
- **품질**: ESLint, Prettier, Vitest, Playwright
- **인프라**: Vercel, GitHub Actions
