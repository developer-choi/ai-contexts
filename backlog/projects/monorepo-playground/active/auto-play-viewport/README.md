# auto-play-viewport (뷰포트 진입 시 동영상 자동재생)

MP `feature/auto-play-viewport` 브랜치에서 작업하던 미머지 실험 코드. 브랜치 삭제 전 보존용.

- 출처 커밋: `f62c33f7` "뷰포트 동영상 테스트중" (2024-12-10)
- 내용: IntersectionObserver로 뷰포트 진입을 감지해 `<Video>`를 자동재생하는 실험.
- 파일은 원본 `src/` 기준 상대 경로 그대로 보존 (`index.module.scss`는 커밋에 포함되진 않았으나 컴포넌트 동반 파일이라 함께 보존).
