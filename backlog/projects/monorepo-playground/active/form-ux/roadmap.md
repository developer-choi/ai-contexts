# Form UX 로드맵

MP의 폼 UX 개선 사례로 발전시킬 항목.

## 해야 할 것

- **폼 베스트 프랙티스 예제 페이지** — [`example-pages.md`](./example-pages.md)
  `/form` 라우트에 공통 규칙(유효성검증·제출·이탈 방지·단축키 등)과 유형별 예제(숫자·주소·파일·비밀번호·회원정보·복합 필드·input type)를 구축. 진행 상태·남은 작업·새 페이지 추가 플로우 포함.

- **검색 폼** — [`search-form.md`](./search-form.md)
  빈 검색 얼럿, 삭제버튼·초기화 유무에 따른 분기, 검색 타입 전환 시 입력값 처리, URL 페이지네이션 연동.

- **검색폼·필터 초기화** — [`검색폼-필터-초기화.md`](./검색폼-필터-초기화.md)
  쿼리 ↔ 입력값 동기화, 필터 초기화 시 폼 값 동반 초기화, 검색·필터의 1페이지 리셋.
