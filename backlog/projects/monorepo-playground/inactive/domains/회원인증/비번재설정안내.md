# 비밀번호 재설정 / 변경 페이지

## 입력폼 동작

- 비밀번호 입력 규칙은 Handle password form 참조
- 기존 비밀번호와 변경할 비밀번호가 같으면 "기존 비밀번호와 동일하다" 노출
- 제출 실패 시 에러메시지 노출 (예: "00일 전에 사용했던 비밀번호는 재사용 못 해요")
- 성공하면 로그아웃 후 로그인 페이지로 이동
- 이후 변경한 비밀번호로 로그인되는지 확인

## 30일 뒤 재안내 버튼

API 호출하고, 성공하면 메인페이지로 리다이렉트.
