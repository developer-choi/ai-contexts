# 내정보 수정 페이지

## 페이지 접근

로그인 + 회원 재확인이 필요한 Private page. (재확인 흐름은 `회원재확인.md` 참조)

## 수정 요구사항

- 각 입력 항목을 하나씩 수정했을 때 정상 수정되는지 확인
  - 주소 수정 시 [기본 배송지에 추가] 체크 후 수정하면 배송지 관리 페이지에 해당 배송지가 추가되는지 확인
- 제출 시 분기 처리
  - 비밀번호를 포함해 수정한 경우: 얼럿 후 로그아웃 처리 → 이후 수정한 비밀번호로 로그인 성공하는지 확인
  - 비밀번호를 제외하고 수정한 경우: "수정되었습니다" 얼럿 처리
