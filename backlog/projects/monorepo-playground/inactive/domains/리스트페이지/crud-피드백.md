# 리스트(목록) CRUD 후 피드백 처리

이 모든 것의 가장 핵심은, "직관적으로 사용자가 한 행동의 결과를 인지시킴"이다.

## Pagination 기반

### 추가

- 최신순 정렬이면서 (네이버 카페 게시판 글목록페이지) > 목록페이지에서 (페이지 이동 없이 모달 같은 수단으로) 추가를 하는 경우 > 1페이지로 이동
- 과거순 정렬이면서 (네이버 블로그 댓글목록) > 목록페이지에서 (페이지 이동 없이 모달 같은 수단으로) 추가를 하는 경우 > 가장 마지막 페이지로 이동
  - 이게 제일 직관적이긴 함. 근데 댓글작성 성공했을 때 목록의 가장 마지막 페이지번호를 API에서 응답할 수 있을지는 잘 모르겠음.
- 목록페이지에서 별도의 추가 페이지로 간 다음 추가가 된 경우 (네이버 카페 게시판 글 쓰기 페이지)
  - 추가한 것을 별도로 볼 수 있는 상세페이지가 있으면 상세페이지로 이동 (ex: 네이버 카페 게시판 글쓰기 성공 후 액션)
  - 그 외에는 목록이 최신순이면 1페이지, 과거순이면 마지막페이지로 이동

### 삭제

- 별도의 상세페이지에서 삭제를 하는 경우 > 1페이지로 이동 (+ 별도의 토스트 메시지)
- 목록 페이지에서 특정 항목을 삭제하는 경우 > 해당 페이지 목록만 새로고침 (1페이지로 보내지 않음)

### 수정 or 기타설정 (ex: 차단)

- 목록페이지에서 해당 액션을 한 경우 > 해당 목록페이지 데이터만 새로고침 (1페이지로 보내지 않음)

## Infinite scroll 기반

### 추가

- 최신순 정렬인 경우 (ex: Youtube 댓글목록) — 이런 경우 댓글작성폼이 목록보다 위에 있음.
  - 방법1. 댓글작성 API 요청 후 성공하면, 사용자 PK와 timestamp 값 같은 거 프론트에서 만들어서 목록 제일 위에 끼워줘도 될 것 같은데… 이렇게 하는 개발자를 못 보기도 했고, 내가 댓글 작성하는 와중에 다른 사람이 댓글 작성한 경우 그 사람 댓글이 안 보인다는 문제가 있긴 함. 그치만 또 사용자가 댓글 작성했을 때 그 사이에 누군가 댓글을 작성했을 거라고 예상하고 댓글 쓰긴 할까? 라고 생각하면 또 애매…함
  - 방법2. 그냥 기존 목록 싹 날리고 (infinite scroll로 인해 목록 누적된 상태) 첫 목록으로 다시 최신화하면 제일 위에 노출될 것.
- 과거순 정렬인 경우 (ex: 네이버 카페 게시판 댓글목록) — 이런 경우 댓글폼은 댓글 목록 가장 하단에 있음 (가장 최근 댓글이 가장 밑에 있음).
  - 방법1. 댓글작성 API 요청 후 성공하면, 사용자 PK와 timestamp 값 같은 거 프론트에서 만들어서 목록 제일 아래에 끼워줘도 될 것 같은데… 애매한 이유는 위와 동일.
  - 방법2. 어쩔 수 없이 infinite scroll 기능으로 인해 목록 갯수가 많아진 상태에서 기존 목록을 다 날리고 첫 목록 데이터를 받아서 사용자가 다시 처음부터 스크롤을 아래로 계속 내려서 본인의 댓글을 확인하는 것이긴 한데 이건 또 너무 불편함.

### 삭제

- 방법1. 목록의 정렬순서와 상관없이 그냥 직접 삭제 API 호출하고 성공하고 나서 목록 API 새로 안 부르고 직접 목록 데이터 하나 삭제해도 될 것 같은데… 애매한 이유는 역시 위와 동일.
- 방법2. 마찬가지로 성공하고 나서 목록 API 새로 호출할 경우, 최신순이든 과거순이든 엄청 불편한 건 똑같음.
  - 최신순이면 한참 스크롤 내려서 저 밑에 있는 내 댓글 삭제하고 나서 다시 목록 초기화되고 또다시 스크롤 한참 내려야 하니까.
  - 과거순이면 한참 스크롤 내려서 저 밑에 있는 … (이하 어차피 동일)

### 수정

추가 / 삭제와 별반 다르지 않음.
- 결국 직접 최신화하던가,
- 목록 API 새로 불러서 새로 스크롤 내려서 최신화된 데이터를 불편하게 확인하던가.

## 핵심 원리

수정 행위는 목록 갯수의 변화를 주지 않기 때문에, 2페이지에서 3번째 항목을 수정했다면, 2페이지 그대로 머무르면서 2페이지 목록 데이터만 최신화해주면, 사용자는 3번째 항목만 데이터가 변경되는 것을 바로 볼 수 있다.

하지만 추가 / 삭제 행위는, 목록 갯수의 변화를 주기 때문에, 사용자가 한 행동의 결과를 바로 볼 수 있는 곳으로 보내주는 것이다.
