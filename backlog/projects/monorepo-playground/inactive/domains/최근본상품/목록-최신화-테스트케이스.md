# 최근본 상품 목록 최신화 테스트케이스

## 최근본 상품 목록 최신화 조건

대상 UI:

- 마이페이지 > 최근본상품 페이지
- (모바일) 아무 페이지 > 전체메뉴 레이어

- 위 [대상 UI]에 최초 접근 시 API 호출해야 함.
  - 마이페이지 > 최근본상품 페이지 진입할 때
  - (모바일) 아무 페이지 > 전체메뉴 레이어 오픈하는 경우
- 위 [대상 UI]에 재접근 시에는 API 호출하지 않아야 함.

### 목록에 추가되는 조건

- 상품 상세페이지에 진입했을 때,
- 상품이 진열 상태이면서,
- 기존 최근본 상품 목록에 없는 상품인 경우에만 목록에 추가되어야 하고,
- 추가될 때 새로 API를 호출하여, 위 [대상 UI]에서 최신화된 최근본 상품 목록을 볼 수 있어야 함.
- 이때, 최근 본 순서대로 목록에 노출되는지 확인해야 함.

### 목록에서 일부를 삭제하는 경우

- API를 호출하지 않고 (= 캐시에서만 삭제),
- 위 [대상 UI]에서 삭제된 상태의 최근본 상품 목록을 볼 수 있어야 함.

## (공통) 체크박스 선택 관련

체크 / 해제 / 삭제 기능:

- 선택삭제, 단일삭제 기능이 동작해야 함.
- 전체선택 체크/해제 시 모든 체크박스 체크/해제되어야 하고,
- 모든 체크박스 체크/해제 여부에 따라 전체선택도 체크/해제되어야 함.
