# `/form/shortcuts` — 키보드 단축키

form-ux 공통 규칙 예제 페이지(신규). **현재 inactive — 진행 보류.** 재개 시 `active/form-ux/`로 promote하고 `example-pages.md` 공통 규칙에 복귀시킨다.

- `Ctrl+Enter` 제출 (특히 textarea — `form.requestSubmit()` 활용)
- `Ctrl+A` 전체선택 (메일리스트, 어드민 테이블 등)
- `Shift` 구간선택 (리스트 아이템 범위 선택)

원문(폼 핸들링 공통 규칙 Google Doc): "Form submit > Ctrl Enter" 링크만 존재. Ctrl+A·Shift 구간선택은 doc 외 추가 항목.
