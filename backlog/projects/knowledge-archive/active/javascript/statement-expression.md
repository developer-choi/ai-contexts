# Statement vs Expression (vs Declaration)

## 현재 이해

- Statement: 실행되는 문장 (if, for, while 등)
- Expression: 값을 반환하는 식 (함수 호출, 연산 등)
- Declaration: 선언 (변수·함수·클래스 선언) — "또 뭐있더라"의 정답 후보

## 미이해 항목

- 셋의 정확한 경계 — 예를 들어 함수 선언 vs 함수 표현식의 차이 (호이스팅 여부 등)
- Expression statement: 표현식을 문장으로 쓰는 케이스
- 실제 면접 질문 패턴
