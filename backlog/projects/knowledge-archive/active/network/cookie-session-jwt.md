# 쿠키 vs 세션 vs JWT

## 현재 이해

- 쿠키: 브라우저에 저장, 도메인 설정 가능 (부모 도메인으로 설정하면 서브도메인에서도 접근)
- 세션: 더 공부 필요
- JWT: 공부 필요

## 미이해 항목

- 세션이 정확히 어떻게 동작하는지 (서버 측 저장? 클라이언트 측?)
- JWT 구조·검증 방법·언제 쓰는지
- 셋의 트레이드오프 (언제 쿠키, 언제 세션, 언제 JWT)

## 관련 메모

- 면접(큐리어슬리 2026-05-30): CORS 질문에서 쿠키 도메인 설정으로 크로스 오리진 상황 우회 시도 언급
  - HttpOnly 쿠키 → 클라이언트 JS에서 읽기 불가. 이 경우 서버에 요청해서 받아오는 방식
