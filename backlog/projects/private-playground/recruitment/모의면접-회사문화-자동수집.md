## 모의면접 시 회사 문화 파일 자동 수집·재사용

- 채용공고 또는 별도 기업문화 소개 페이지가 있는 회사 한정으로, recruitment skill 면접 준비 첫 step에서 culture.md 자동 조사 및 산출물 생성
- 이후 모의면접 단계에서 해당 파일 자동 재사용
- 좋은 회사는 문화를 별도 안내하지 않으니, 채용공고나 공식 기업문화 페이지가 명시적으로 있는 경우에만 적용
- 현재: 사용자가 직접 파일 경로 안내해야 culture 파일 활용 가능
- 개선 목표: AI가 첫 step에서 스스로 조사 → 산출물 생성 → 모의면접 시 자동 로드
