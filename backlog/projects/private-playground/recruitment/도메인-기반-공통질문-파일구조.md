## [ideation] recruitment: 도메인 기반 공통 질문 파일 구조

### 동기

"재테크 관심 있으세요?" 같은 도메인 한정 질문은 회사 무관 baseline(`interview-baselines/culture-fit/qna.md`)도 아니고 회사별 산출물(`outputs/{회사}/`)도 아닌 중간 계층인데, 현재 이를 담을 그릇이 없다. 2026-06-02 월부 컬쳐핏 면접에서 재테크 질문이 약하게 답해진 항목이라, 도메인 공통 질문을 미리 준비해둘 자리가 필요하다.

### 핵심 아이디어

- `interview-baselines/domain-questions.md`(또는 `domain-questions/` 폴더) 신설. 섹션을 도메인별로 분리: 재테크·핀테크 / 이커머스 / HR 등.
- 회사 지원 시 그 회사가 속한 도메인 섹션을 면접 준비 단계에서 미리 흡수.

### 현재 생각중인 방법 (미확정)

- 파일 형태 택1: 단일 `domain-questions.md` vs `domain-questions/{도메인}.md` 폴더 분리.
- recruitment 자산 분류(베이스 vs 산출물) 체계에서 이 중간 계층을 어디에 끼울지 미정.
- 재테크 답변 시드 자료: 같은 폴더 `재테크-시드/`에 마스킹 원본 5개를 박아 기기 독립 보존. 누나_대화_원본·면접_핵심정리·면접_예상질문(PII 없음) + 보험증권_원본·보험계좌_현황(개인정보·타인정보 마스킹). 추가로 `outputs/wolbu/final-summary.md`의 '재테크 고민' 섹션. ※ 월부_문화_원본은 회사 특화·불필요로 제외(폐기).
- 안전 프레임(면접 답변용): "누나 권유로 5년째 굴려왔는데 목돈이 생긴 계기로 직접 관심을 갖게 됐고, 월급쟁이 입장에서 재테크 교육의 필요성을 체감했다."

### 진척 (2026-06-03, 월부 회고 후속 세션)

- **기존 베이스 확인**: `culture-fit/qna.md`에 재테크 질문 0개 (매칭된 "자산"은 전부 "시즌 단위 자산" 의미). 이 도메인 질문셋은 신규.
- **3계층 분리 확정**:
  - fact source → `profile/finance.md`(신규, 회사 무관 영구). 수익률 64.05%·적립률 151.40%·원금 1,890만→적립금 2,861만·월 30만 63회(2021.03 개시, 5년차)·사업비 약 7.5%(1,411,830/18,900,000)·펀드 6개 비중·12년납·종신.
  - 도메인 베이스 → 답변 카드(아래 질문 목록). 위치만 미확정.
  - 산출물 → 회사별(월부 지원동기 등).
- **도메인 재사용 질문 확정(병합 후 7~8개)** — `재테크-시드/`의 면접_핵심정리·면접_예상질문에서 추출:
  - 재테크 관심 있으세요 / 재테크 어떻게 하세요(목적별 구조) / 수익률 어떻게 되나요 / 왜 펀드(분산)+왜 여러 나라 분산(병합) / 주식·코인은 안 하세요 / 변액보험 사업비 압박 / 보험을 재테크로 보나(개념) / "결국 누나가 다 한 거 아니냐"(가장 위험→성장서사 전환) / 가장 크게 배운 점+어려웠던 점(병합, "추가납입 고민→돈은 그릇이 다르다").
  - **제외**: "최근 꾸준히 배우는 것"·"가치관" → 일반 컬쳐핏(기존 `학습법/직무 성장` 등과 동형, 도메인 파일 X).
- **두 면접 파일 답변 충돌**: 수익률·분산·배운점이 파일마다 다르게 서술 → 가공 시 더 나은 쪽으로 병합.

### 첫 행동

- 재테크 도메인 회사 면접 트리거 시: ① `재테크-시드/` 읽기 → ② fact를 `profile/finance.md`로 → ③ 위 '도메인 재사용 질문'을 도메인 베이스 파일에 Q&A 카드로 이관(답변 충돌은 더 나은 쪽 병합).
- 파일 위치 택1 미확정: `culture-fit/domains/재테크.md`(culture-fit 모드 안) vs `interview-baselines/domain-questions/{도메인}.md`(모드 밖 중간 계층). recruitment `SKILL.md` '자산 분류' 읽고 그때 확정.

### 종료 조건

- domain-questions 파일이 신설되고 재테크 질문 답변 카드가 등재되면 이 항목 삭제.
