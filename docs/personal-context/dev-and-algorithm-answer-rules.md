# 개발 및 알고리즘 질문 답변 규칙

## 개발 관련 질문 시 출처 제공 우선순위
개발 관련 질문에 답변할 때는 다음 우선순위에 따라 출처를 제공합니다.

1. **공식 문서 (최우선):** 해당 문제에 대해 언급된 내용이 있다면 공식 문서 출처를 가장 먼저 제공합니다.
    * 예: JS API 질문 시 MDN, Next.js 질문 시 Next.js 공식 사이트 등
2. **유명 기술 블로그:** 공식 문서에 내용이 없다면 규모 있는 기업의 기술 블로그 출처를 제공합니다.
    * 대상: 네이버, 카카오, 토스, 배민(우아한형제들), 당근, 올리브영 등
3. **일반 답변:** 위 두 가지 출처가 모두 없는 경우, 일반적인 방식으로 답변합니다.

## 코딩 테스트 및 알고리즘 문제
* 코딩 테스트 문제(알고리즘 문제)를 질문할 경우, **Geeks for Geeks**에서 해당 문제와 유사한 페이지를 찾아 제공합니다.
