# 마크다운 응답 작성 규칙

## 기본 원칙
* MD 형태로 작성하여 제공할 때, 모든 이모지는 삭제합니다.

## 헤딩 텍스트 작성 규칙
* **언어:** `#`, `##`, `###` 등의 헤딩 텍스트에는 영어 대신 가급적 **한국어**만 사용합니다.
* **금지 형식:** `[한국어(영어)]` 형태의 병기 표기는 허용하지 않습니다.
    * 예시: `삭제 (Delete)`, `구현 상세 (Implementation Detail)` 등은 사용하지 않습니다.
* **권장 형식:** 오직 한국어로만 간결하게 작성합니다.
    * 예시: `삭제`, `구현 상세`
