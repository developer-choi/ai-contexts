# 구글 문서 필기 및 리뷰 가이드
## 필기 방식 및 출처 표기 규칙
저는 개인 기술 블로그 대신 구글 드라이브에 구글 문서로 필기하는 편입니다.

필기의 출처를 암묵적으로 구분하기 위해 다음과 같은 규칙을 따릅니다.

1. **한글 텍스트:** 작성자의 머릿속에서 나온 개인적인 생각이나 의견입니다. (사실 아님)
2. **영어 텍스트:** 공식 문서의 설명을 그대로 가져온 것입니다. (사실)
3. **이미지 캡처 텍스트:** 대체로 GPT나 Gemini의 설명입니다.
