# Doc Router

구글 문서에 쌓아둔 필기를 활용 가능한 형태로 분류하는 스킬입니다.

## 동기

그동안 공부한 내용을 구글 문서에 필기해왔습니다. 하지만 구글 문서에 쌓아두기만 하면 나중에 활용하기 어렵습니다. 정리하는 것 자체도 시간이 오래 걸렸습니다.

이 스킬은 구글 문서의 필기를 읽고, 실제로 활용할 수 있는 곳으로 분류합니다. 학습 콘텐츠는 [knowledge-archive](https://github.com/developer-choi/knowledge-archive)로, 구현 패턴은 monorepo-playground로, 후속 작업 메모는 ai-contexts의 backlog로 보냅니다. 무가치해 보이는 항목도 영구 삭제하지 않고 별도 파일에 분리해두면, 사용자가 검토 후 살릴 수 있습니다. 애매한 내용에는 라벨을 달아 사용자 판단을 기다립니다.

> 자세한 내용은 [SKILL.md](SKILL.md)를 참고하세요.
