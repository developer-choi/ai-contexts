# Tech Trends

생소한 기술 트렌드를 프론트엔드 개발자 관점에서 설명하는 스킬입니다.

## 왜 만들었나요?

인프라, 백엔드, 데이터 엔지니어링 등 프론트엔드 바깥의 기술을 접할 때, 개념은 이해하고 싶지만 배경 지식이 다릅니다. 일반적인 비유("도서관의 사서처럼...")보다는, 제가 이미 아는 프론트엔드 스택에 빗대어 설명을 들으면 훨씬 빠르게 이해할 수 있었습니다.

이 스킬은 기술명이나 아티클 URL을 주면, 해당 기술이 해결하려는 문제를 먼저 파악하고, 그 해결 전략을 React, Next.js, TypeScript 같은 프론트엔드 패턴에 연결하여 설명합니다.
