# 면접 직후 회고

면접 끝난 직후 AI가 회고를 가이드한다. 사용자가 능동적으로 회고하지 않고, AI가 빈 칸을 메우게 만드는 것이 핵심.

## 절차

1. 클로바노트 녹음 텍스트가 있으면 사용자에게 요청해 받는다
2. 현재 `recruitment/` 폴더 구조·베이스 파일을 다 훑어 **그 시점 기준의 회고 질문지를 동적으로 생성**하고 `outputs/{회사명}/retrospective-questions.md`로 출력. 질문 형식·범위는 AI가 그때그때 판단 (규칙 박지 않음)
3. 사용자가 그 파일에 직접 답변 작성 후 "답변 끝" 신호
4. AI가 답변을 읽고 적절한 위치(베이스 파일 / `company-analysis/` 산출물 / SKILL.md / 백로그)에 반영. 어디 반영할지는 항목별로 사용자 컨펌 후 진행
