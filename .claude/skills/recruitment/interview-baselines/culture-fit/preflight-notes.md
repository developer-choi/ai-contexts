# 면접 직전 메모

> 면접 들어가기 직전에 가볍게 훑는 메모. 자세한 답변 연습은 culture-fit-qa.md.

## 꺼내지 말 키워드/표현

- **"리스펙"** — PR 리뷰 집요 관련 답변 중 사용 금지
- **"문서에 누락이 많아서"** 류 남탓 표현 — 완성도/시간 단축 관련 답변 중 사용 금지
- **"나한테 남는 게 없다"** — 임원면접 전반 사용 금지
- **"커리어"** — 키워드 자체 임원면접 전반 사용 금지

## 꺼내지 말 행동

- **"마지막으로 하고 싶은 말"에 길게 답하지 않기** — 무난해 보이는 질문이지만 길게 끌수록 누적된 평가가 마지막 인상으로 덮인다. 특별히 전할 메시지가 없으면 짧은 감사 인사로 마무리.
