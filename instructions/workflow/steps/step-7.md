# Step 7: PR 본문 작성

Step 6 완료 후, PR 본문을 작성하는 단계입니다.

---

## Step 7.1. PR 본문 작성 (`/plan/pr{N}/pr-body.md`)

[문제 해결 로드맵](../../self-help/roadmap.md)의 5단계를 **사고 프레임워크**로 활용합니다. 고정 양식이 아니라, 각 단계의 관점에서 사용자의 문제 해결 과정을 끌어내고 PR 본문으로 구조화합니다. 세션에서 step 1~6을 함께 진행하며 쌓인 맥락을 활용하되, 사용자가 놓치고 있는 단계가 있으면 질문으로 끌어냅니다.

---

## Step 7.2. 작성 전 체크리스트

- **문서 톤 규칙 로드**: `instructions/coding-standards/writing/` 파일들을 읽기
- **이전 PR 본문 참고**: 이전 PR의 `pr-body.md`를 읽고, 이미지 배치·섹션 구조·서술 패턴을 맞추기
- **말투 일관성**: 작성 완료 후 전체를 훑어 음슴체/습니다체 혼용이 없는지 확인

---

## Step 7.3. 비판적 검토

- **[CRITICAL]** 별도 요청이 없어도 항상 비판적으로 검토해야 합니다.

---

## [CRITICAL] 사용자 확인 필수

> "pr-body.md 작성 완료. 확인해주세요."
