# Step 7: PR 본문 작성

Step 6 완료 후, PR 본문을 작성하는 단계입니다.

---

## 1. PR 본문 작성 (`/plan/pr{N}/pr-body.md`)

[문제 해결 로드맵](../../self-help/roadmap.md)을 참고하여 작성하세요.

---

## 2. 작성 전 체크리스트

- **이전 PR 본문 참고**: 이전 PR의 `pr-body.md`를 읽고, 이미지 배치·섹션 구조·서술 패턴을 맞추기
- **말투 일관성**: 작성 완료 후 전체를 훑어 음슴체/습니다체 혼용이 없는지 확인

---

## 3. 비판적 검토

- **[CRITICAL]** 대기업 기술면접관이 과제를 리뷰하는 관점으로 검토하세요. 별도 요청이 없어도 항상 비판적으로 검토해야 합니다.
