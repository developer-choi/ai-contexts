# 개요
프로젝트 코딩 컨벤션과 리뷰 규칙을 정의한 폴더입니다.

코드 작성, 테스트, 리뷰 등 개발 작업 시 참고하세요.

---

## 📖 폴더 구조

### essentials/
프로젝트 핵심 가치 및 필수 점검 사항입니다.

**모든 작업에 필수로 읽어야 합니다.**

### coding/
언어 및 프레임워크별 작성 규칙입니다.

**기능 구현, 버그 수정, 리팩토링 시 참고하세요.**

하위에 공통 규칙과 TypeScript, React, 마크업 관련 규칙이 있습니다.

### testing/
테스트 코드 작성 규칙입니다.

**테스트 작성 시에만 참고하세요.**

### review/
코드 리뷰 및 PR 작성 가이드입니다.

**코드 리뷰 또는 PR 작성 시 참고하세요.**
